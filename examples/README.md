# PotentialFieldEngine 예제

이 폴더에는 PotentialFieldEngine의 사용 예제와 데모가 포함되어 있습니다.

## 📖 예제 목록

### test_demo.py
기본 테스트 및 데모:
- 중력장 데모
- 그리드 분석 데모
- 왜곡 탐지 데모

**실행 방법**:
```bash
python3 test_demo.py
```

### demo_analytical_field.py
해석적 필드를 사용한 궤도 운동 데모:
- 원형 궤도 운동 시뮬레이션
- 타원 궤도 운동 시뮬레이션
- 에너지 보존 확인
- 궤도 반지름 일정성 확인

**실행 방법**:
```bash
python3 demo_analytical_field.py
```

**검증 항목**:
1. 짝수 길이 규약 강제 확인
2. 해석적 필드 사용 확인
3. 궤도 운동의 물리적 정확성
4. 에너지 보존 확인
